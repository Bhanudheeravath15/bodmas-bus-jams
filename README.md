# BODMAS Bus Jam 🚍

An interactive educational game that helps learners understand
BODMAS (order of operations) through visual interaction.

## 🎮 Game Features
- Users can enter their own numbers
- Supports operators: +  -  ×  ÷
- Players click operators in correct BODMAS order
- Instant visual feedback for correct and wrong choices

## 🧠 Learning Objective
To help students understand operator precedence visually
instead of memorizing rules.

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- PHP

## ▶️ How to Run Locally
```bash
php -S localhost:8000
``
